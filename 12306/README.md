## 12306插件

###功能
- 表单自动填充（不需担心被挤下线表单数据丢失）
- 订单自动提交（提高订单提交速度）

###使用
- 填写配置信息并保存
- 使用chrome浏览器打开[12306官网](http://www.12306.cn/)，并进入购票页面
- 在登录弹层确认验证码，完成登录操作
- 自动刷票完成，订单提交时，只需确认验证码即可确认订单

###说明
本插件使用[localStorage](https://en.wikipedia.org/wiki/Web_storage#localStorage)保存用户配置信息，不会收集账号和密码
